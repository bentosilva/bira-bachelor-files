Enunciado do Projeto

Implemente um sistena de e-Commerce que mantenha informações sobre os clientes, compras, produtos comprados, produtos disponíveis. Cada produto pode ter múltiplas categorias. Deve-se: tratar sessão de login, internacionalização, utilizar modelo MVC, um banco de dados a sua escolha, Apache Ant, upload de imagem do produto, utilizar criptografia para senha de usuário (mas deve ser diferente de MD5, a qual foi usada em sala de aula).

[Obs.] Data de entrega estipulada em 19 de junho de 2011.

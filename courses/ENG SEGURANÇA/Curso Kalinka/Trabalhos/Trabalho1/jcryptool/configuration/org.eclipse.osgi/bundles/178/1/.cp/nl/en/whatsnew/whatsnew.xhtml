Algorithms View

The Algorithms view shows on four tabs - Algorithms, Analysis, Games and Visualizations - all currently installed crypto plug-ins in JCrypTool. Simply double click on an entry to start the desired wizard or open the plug-in's view. More...

Actions View

The Actions view enables you to record most of the cryptographic operations available in JCrypTool. While recording, all required information on the executed cryptographic operations are automatically collected. The generated protocol can be executed again. Additionally it can be saved and shared with other users. More...

Cryptography Plug-ins

A lot of crypto plug-ins, ranging from Algorithms, Analysis, to Games up to Visualizations, are waiting for you.

Help and Context Help

The JCrypTool help does contain a lot of information for JCrypTool core and most crypto plug-ins. Additionally it provides background information about different cryptographic operations. At the same time more plug-ins are now supporting the context sensitive help view and are providing short context information while executing a certain action or selecting an element in the application.

What is JCrypTool?

JCrypTool is the cryptography e-learning platform. The complete platform is developed as an open source project and is available at no costs for all users. It is JCrypTool's aim to interest people for cryptography and to offer them an easy to use and modern tool to try out different cryptographic methods.
More information on JCrypTool's possibilities and their usage is available in the online help in the chapter User Guide.

How can I learn something about cryptography with JCrypTool?

JCrypTool offers easy access and enables experiments with various cryptographic functions on arbitrary files with the help of different wizards and dialogs. The user is guided step by step through every process. Background information about the history or the technical details of an algorithm or a cryptographic analysis is available in the online help in the chapter Cryptology.

How can I extend JCrypTool with my own cryptographic plug-in?

JCrypTool is based on the Eclipse Rich Client Platform (RCP) and can therefore be extended in almost any place. To provide a unique and easy to use access to all cryptographic functionality for any user, JCrypTool offers different extension points which must be used by all cryptographic plug-ins. This way, cryptography plug-ins can offer encryption algorithms, digital signatures, cryptanalysis or visualizations, which then appear at once in a common way like the in the Algorithm view.
More information on developing plug-ins is available in the online help in the chapter Developer Guide.
